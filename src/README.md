# 中部状态

非操作区， 主要展示播放器当前的操作信息

## 实现

1. 分为两种状态

    * 播放正常中的一些状态

        * loading （z-index 低）

        * 提示用户的一些状态（快进，后退，音量增， 音量减， 播放， 暂停）

    * 错误（与正常状态互斥）
